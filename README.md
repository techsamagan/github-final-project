# calculator
📌 Simple Interest Calculator

This project is a Simple Interest Calculator developed to help users easily calculate the interest earned on a principal amount over a specific period of time at a given interest rate.

⸻

🚀 Features
	•	Calculate simple interest quickly and accurately
	•	Easy-to-use and beginner-friendly
	•	Lightweight and efficient
	•	Can be run on any system with basic setup

⸻

🧮 Formula Used

The calculator uses the standard formula:
Simple Interest = (Principal × Rate × Time) / 100
Where:
	•	Principal (P) = Initial amount of money
	•	Rate (R) = Interest rate per year
	•	Time (T) = Time period in years
